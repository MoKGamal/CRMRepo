Map Test Launcher

Paste or edit pin groups JSON below, then choose how to open the map.

Expected Structure:

Each pin group has a color (hex) and an array of pins:

[
  {
    "color": "#FF0000",
    "pins": [
      {
        "lat": 47.5162,
        "lng": 14.5501,
        "info": "Info text with <br> tags",
        "azimuth": 60,
        "zoom": 7,
        "sequence": "1,4,5"
      }
    ]
  },
  {
    "color": "#00FF00",
    "pins": [ ... ]
  }
]

Colors: #FF0000 (Red), #00FF00 (Green), #0000FF (Blue), #FFFF00 (Yellow), #FF00FF (Magenta), #00FFFF (Cyan), #FFA500 (Orange)
